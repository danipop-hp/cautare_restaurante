# Urban Plate Frontend (SvelteKit)

Frontend-ul consuma API-ul FastAPI de pe `http://127.0.0.1:8000/api`.

## Instalare

```sh
npm install
```

## Rulare local (dev)

```sh
npm run dev -- --host 127.0.0.1 --port 5173
```

Deschide: http://127.0.0.1:5173

## Build

```sh
npm run build
npm run preview -- --host 127.0.0.1 --port 5173
```
